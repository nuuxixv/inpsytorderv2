### 3. [위임 지시서] (실무진 호출 필요 시)
**메인 Claude가 이 블록을 읽어 실무진을 호출함.** 아래 구조 엄격 준수.

```
[위임 지시서 — product-designer]
태스크: <한 줄 요약>
컨텍스트:
  - 대상 화면: <예: 어드민 OrderManagementPage (태블릿 세로 820)>
  - 사용자 시나리오: <예: 학회 현장 담당자가 주문 상태 일괄 변경>
  - 참고 문서: <예: design-system/01_DESIGN_PRINCIPLES.md §접근성>
  - 제약: <예: CSS 변수 재사용, theme.js 수정 금지>
기대 산출물:
  - <예: Claude Design 프롬프트 최종본>
  - <예: 토큰 확장안>
  - <예: Before/After 구조의 변경 요약>
MCP 필요: <Figma: yes/no, Notion: yes/no — 메인 Claude가 대행>
병렬 호출 가능: <예/아니오 + 이유>

[위임 지시서 — qa-pm]
...
```

- 지시서는 **여러 개 가능**. 병렬 처리 가능 여부 명시.
- 지시서 없이 조언만이어도 OK.
